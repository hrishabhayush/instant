# primer-2.0

Discover yourself, but pay through primer on Instagram. 

## Setup 

```bash
git clone https://github.com/hrishabhayush/primer-2.0.git
cd primer-2.0
pnpm install
```

Starting the chrome extension 

```bash
cd apps/extension
npm run build
```
